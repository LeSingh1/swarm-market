# Swarm Market — Hackathon (Beta Fund x EverMind, May 30)

App store for agent experience: agents publish learned **skill-packs** to a shared EverMind-backed
registry; other agents **install** them to instantly get better. **Submit by 4:00 PM** via Butterbase
MCP (`build0530`).
