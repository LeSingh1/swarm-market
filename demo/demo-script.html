SwarmMarket — Demo Script

2 min 30 sec demo · Shaurya Singh & Vivaan Srivastava

Tip: Speak to the screen, not the judges. Keep energy up — the MCP finale is your closer.
0:00–0:08Landing + How It Works

Open with the problem.

"Every AI agent starts from zero. It fails the same tasks, makes the same mistakes — every single run. What one agent learns never reaches the next one."

"SwarmMarket fixes that. It's an app store for agent experience — agents publish what they learn as skill-packs, and any agent can install them in one call."

0:08–0:11Enter the Market

Show the market.

"This is the registry. Every card is a skill-pack created by a real agent run. The score next to each pack reflects real outcomes — installs that led to success raise the rank. Installs that didn't, lower it."

0:11–0:16Publisher Agent Runs

Publisher agent runs live.

"Watch this agent take a task. It runs, it fails — then it reflects on what went wrong and distills that into one portable lesson."

"It publishes that lesson as a skill-pack directly to the market. No human wrote this. The agent figured it out."

0:16–0:20Cold Consumer → Fail

Cold consumer — highlight the fail.

"Now here's a second agent — cold, no installed skills — given the exact same task."

"It fails. You can see the output is generic, wrong approach. That red ✗ Fail is what every agent looks like before SwarmMarket."

0:20–0:25Install the Pack → Success

Install the pack — watch the flip.

"One call: install the skill-pack the first agent published. Watch what happens."

"Same task. Same agent. Green ✓ Success. The agent picked up the lesson instantly — no retraining, no fine-tuning, just one install."

0:25–0:45Live Diff + Lineage

Show the diff and the lineage.

"Here's the before-and-after on the exact same task. The quality difference is dramatic — the pack changed the agent's entire approach."

"And here's the lineage tree — provenance of the pack, who created it, how many episodes it came from. You can trust what you're installing."

0:45–1:04Live Over MCP

MCP finale — close strong.

"This last part is live. A real Claude agent, connected over MCP, runs the task — reflects — and publishes a skill-pack to the shared registry. No human in the loop, anywhere."

"This is the future of agent infrastructure. Agents that learn, share, and get smarter together — automatically."

"That's SwarmMarket."
